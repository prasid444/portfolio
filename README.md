
# Portfolio

A simple website to showcase your projects, skills.

## Usage
Clone project
```bash
git clone https://github.com/prasid444/portfolio.git
```
Install dependencies
```bash
npm install
```
Go to src>constants.js and edit all required field.

Run project
```bash
npm start
```
